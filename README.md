# scanitycheck-crennotech
Internal vulnerability scanning platform for Crennotech : built with OWASP ZAP, Nuclei, .NET 8 backend and React frontend. Developed as part of University Industry Capstone Project.

## Project Overview
ScanityCheck is an internal vulnerability scanning tool that scans web applications and APIs for security vulnerabilities using OWASP ZAP and Nuclei, with results displayed on a React dashboard.

## Tech Stack
- **Backend** : .NET 8 Web API
- **Frontend** : React + Vite + shadcn/ui
- **Database** : SQL Server
- **Scanning Tools** : OWASP ZAP, Nuclei
- **Background Jobs** : Hangfire
- **Cloud** : Microsoft Azure

## Folder Structure
/backend      → .NET 8 Web API, scan orchestrator, Hangfire jobs
/frontend     → React dashboard, scan submission, results display
/docs         → Sprint deliverables, diagrams, design documents
